Introdução a GitHub - Nível básico

GitHub é uma plataforma de hospedagem de repositórios e arquivos de controle de versão que utiliza o sistema Git como base. Ele fornece diversas ferramentas aos desenvolvedores cadastrados, permitindo que contribuam em projetos privados ou de código-aberto de qualquer lugar do mundo. Neste site, ensinaremos algumas funcionalidades básicas desta plataforma extremamente importante e popular entre os programadores.
